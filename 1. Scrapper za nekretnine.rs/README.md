# Skrejper za nekretnine

Python modul za prikupljanje oglasa o nekretninama sa sajta nekretnine.rs ( moze se prilagoditi i za druge sajtove) i cuvanje podataka u CSV fajlu

## Instaliranje i pokretanje
prvo-
pip install -r requirements.txt
pa-
python -m scraper.scraper


Podaci se cuvaju u `data/raw/dataset.csv`
